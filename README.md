## 📌 Descripción del Proyecto
InnovaApp es una solución integral desarrollada para optimizar la cadena de valor de una fábrica de sofás. El proyecto nació de la necesidad de digitalizar un proceso manual, centralizando la toma de pedidos, el seguimiento de fabricación y la gestión financiera en una sola interfaz ágil y profesional.

## 🚀 Problemas Reales, Soluciones Digitales
Este desarrollo no es solo código; es la respuesta a desafíos operativos concretos:
* **Desorden en Pedidos:** Centralización en una base de datos relacional (**Supabase**) eliminando el uso de anotaciones dispersas.
* **Fuga de Capital:** Control estricto de señas y saldos pendientes con alertas visuales de "Restan cobrar".
* **Cuellos de Botella en Producción:** Un "Tablero de Producción" con semáforo de días para priorizar pedidos urgentes.
* **Burocracia Manual:** Generación automática de **Remitos en PDF** y acceso directo a **WhatsApp** del cliente en un clic.

## ✨ Funcionalidades Clave
- **Dashboard de Caja:** Visualización en tiempo real de ingresos totales, señas cobradas y dinero "en la calle".
- **Gestión de Clientes:** Módulo diferenciado para ventas minoristas y **Área Mayorista**.
- **Logística de Producción:** Expanders inteligentes que muestran tiempo en taller, detalles de producto (color, notas) y estado del pedido.
- **Filtros Avanzados:** Buscador dinámico por nombre, color o detalle del pedido.

## 🛠️ Stack Tecnológico
- **Lenguaje:** Python 3.x
- **Frontend:** Streamlit (UI/UX enfocada en eficiencia).
- **Backend/DB:** Supabase (PostgreSQL).
- **Librerías destacadas:** `fpdf` (generación de documentos), `pandas`, `datetime`.
